🎓 PhD Student Enrollment Analysis Dashboard

🚀 Overview

The **PhD Student Enrollment Analysis Dashboard** is an executive-level **Power BI analytics solution** designed to monitor doctoral student enrollment, academic progression, and fee payment compliance across multiple universities and institutes.

The dashboard provides centralized visibility into **student status, academic pipeline progression, financial compliance, supervisor workload, gender diversity, and institutional risk factors**. It enables university management and administrators to identify **high-risk fee defaulters, enrollment trends, and institutional performance** through interactive visualizations and advanced analytics.

---

🎛️ Dashboard Filters

The dashboard allows dynamic filtering across multiple academic and institutional dimensions:

1. Academic Year
2. University
3. Institute Location
4. Branch
5. Domain
6. Degree
7. Gender
8. Institute

These filters enable users to drill down into **specific institutions, academic batches, domains, and demographic segments**.
