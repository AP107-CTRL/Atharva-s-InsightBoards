💰 Cash Flow BI Dashboard

🚀 Overview

The Cash Flow BI Dashboard is a Power BI project designed to monitor and analyze cash expenses across multiple college campuses under a single educational organization.

This dashboard provides a structured financial overview through day-wise and month-wise analysis, enabling management to track expenses, compare financial years, and monitor cost distribution efficiently.

📂 Dataset Scope

The dashboard analyzes financial transaction data including:

Date
Financial Year (FY)
College
Account Head
Cash Receiver
Expense Amount
Quarter
Month
Day
📊 Dashboard Structure

The report is divided into two main analysis tabs:

📅 1️⃣ Day-Wise Analysis Tab

🎛️ Filters Available

Amount ≥ (Dynamic threshold filter)
Date
Month
Financial Year (FY)
Colleges
📈 Visualizations

💳 Total Cash Expenses Across Financial Years (KPI Card)
📊 FY Year-wise Total Cash Expenses (KPI Card)
📑 Daily Account Head-wise Expenses
Includes: Account Heads, FY, College, Month, Day

🧾 Daily Cash Receiver-wise Expenses

Includes: College Name, FY, Month, Day
This tab allows detailed tracking of daily cash outflows across campuses.

📆 2️⃣ Monthly Comparison Tab

🎛️ Filters Available

Amount ≥ (Dynamic threshold filter)
Date
Month
Financial Year (FY)
Colleges
Quarters
📈 Visualizations

📊 Cost by Months

Includes: Months, Financial Years, Monthly Cost Values
📈 Cost by Year & Quarters

Includes: Financial Years, Quarters, Months, Total Cost
This tab enables comparative financial analysis across months, quarters, and financial years.
